# 猫叔有段位•星航 — 个人导航网站 PRD

## 1. 产品概述
个人导航网站，帮助用户集中管理和快速访问常用网站链接，支持分类模块化管理、搜索过滤、主题切换和管理后台配置。
- 目标用户：个人用户，需要一个美观、高效的书签导航页
- 核心价值：Sun-Panel 风格的深色玻璃质感仪表盘，极简图标化卡片，一键部署到 Cloudflare Pages

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览导航、搜索、切换主题 |
| 管理员 | 首次设置密码 | 全部管理功能（CRUD、配置、导入导出） |

### 2.2 功能模块
1. **导航主页**：顶部标题区 + 搜索框 + 侧边栏导航 + 模块化网站卡片
2. **管理后台**：登录认证、站点设置、模块/链接 CRUD、数据导入导出

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 导航主页 | 顶部区域 | 居中标题、实时日期时间、搜索框（支持搜索引擎切换和卡片过滤） |
| 导航主页 | 侧边栏 | 模块导航列表（点击滚动定位、当前高亮）、主题切换、管理入口 |
| 导航主页 | 主内容区 | 模块卡片容器（可折叠）、网站卡片（Favicon+名称）、拖拽排序 |
| 管理后台 | 登录面板 | 首次设置密码 / 密码登录，获取 JWT |
| 管理后台 | 设置面板 | 站点标题/字体/字号、日期格式、背景图、搜索引擎、主题、导入导出 |

## 3. 核心流程

### 3.1 访客浏览流程
用户打开网站 → 查看导航卡片 → 可搜索过滤 → 点击卡片新标签打开网站 → 可切换主题

### 3.2 管理员操作流程
管理员点击齿轮图标 → 输入密码登录 → 进入管理状态 → 可编辑/删除/新增模块和链接 → 修改全局设置 → 退出管理

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深色主题为默认，背景 #0f172a，卡片 rgba(255,255,255,0.05)，强调色 #6366f1（靛蓝）
- **辅助色**：白天模式 #f8fafc 背景，护眼模式 #1a1a2e 背景
- **按钮风格**：圆角半透明玻璃质感，hover 时亮度提升
- **字体**：标题使用 "Outfit" 优雅无衬线字体，正文使用系统字体栈
- **布局风格**：顶部通栏 + 左侧固定侧边栏 + 右侧自适应主内容区
- **图标风格**：lucide-react 线性图标，模块标题使用 emoji

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 导航主页 | 顶部区域 | 深色半透明背景、大号标题居中、实时时钟、圆角搜索框、搜索引擎切换按钮 |
| 导航主页 | 侧边栏 | 磨砂玻璃效果、模块图标+名称列表、当前项靛蓝高亮、底部主题/齿轮按钮 |
| 导航主页 | 主内容区 | 模块圆角卡片、emoji+标题行、可折叠、网站卡片网格、48px Favicon+名称 |
| 管理后台 | 登录面板 | 居中模态框、密码输入框、登录按钮 |
| 管理后台 | 设置面板 | 右侧滑出面板、分组设置项、输入框/下拉/颜色选择器 |

### 4.3 响应式设计
- 桌面端（≥768px）：左侧侧边栏 + 右侧主内容区
- 移动端（<768px）：侧边栏隐藏，底部固定导航条，卡片两列网格，顶部区域缩小

### 4.4 动效设计
- 页面加载：卡片 stagger 渐入动画
- 悬停：卡片微放大 + 阴影增强，编辑/删除图标淡入
- 主题切换：CSS 变量平滑过渡（transition 0.3s）
- 搜索过滤：卡片淡入淡出
- 侧边栏导航：滚动高亮跟随
